# BeeWise

A spelling practice application built with pygame.
This program was created as the final project of COMP9001 S2-2025.

## Installation

Install required dependencies:

```bash
pip install pygame pyttsx3 rapidfuzz
```

## Setup

The `words_mixed_levels.csv` file must be in the root directory of the project.

The program will create directories (such as `users/`) in the root directory of the project.

## Running

Run the application:

```bash
python beewise.py
```

## Tests

Run the tests:

```bash
python test_word_selection.py
```
